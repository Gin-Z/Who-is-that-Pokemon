Proyecto hecho con React + Typescript.
El usuario debe ingresar el nombre de un Pokemon en el input y hacer clic en el botón para ver si su respuesta es correcta o no.
La aplicación mostrará la imagen del Pokemon, con su respectivo nombre e informará al usuario si acertó con el nombre o no.

Se utilizaron: useEffect, useState, useCallback, useRef, hooks personalizados, APIs de terceros, librerías de estilos externas, fetching de datos, props e interfaces.

APIs de terceros utilizadas: https://pokeapi.co/api/v2/pokemon/
Librerías de estilos utilizadas: Bootstrap 5, React Confetti, Boostratp Icons
